# Stock Portfolio Tracker

## Description
A simple Python program that calculates the total investment value of stocks based on user input.

## Features
- User enters stock name
- User enters quantity
- Calculates total investment value
- Uses predefined stock prices

## Technologies Used
- Python

## How to Run
1. Open stock_tracker.py in VS Code.
2. Run the program.
3. Enter stock name and quantity.
4. View the total investment value.
